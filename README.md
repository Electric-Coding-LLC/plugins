# Electric Coding Plugins

Open-source plugin collection maintained by Electric Coding LLC.

This repo holds individual plugin folders that can be copied into a local plugins directory and configured on your machine.

## Available Plugins

- [`google-stitch`](./google-stitch): Community-maintained Codex plugin for Google's hosted Stitch MCP server. Unofficial and provided as-is.

## Repo Layout

- `google-stitch/`: plugin files, setup notes, and local MCP template
- `LICENSE`: shared MIT license for the repo

## Usage

Open the README inside the plugin folder you want to install. Each plugin directory contains its own setup instructions and any local configuration templates it needs.
